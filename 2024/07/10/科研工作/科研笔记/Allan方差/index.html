# 回顾传统误差指标

\quad 考虑要测量一个如图 111 所示的起伏时间序列函数 y(t)y(t)y(t)，我们用测量器读出 NNN 个离散的数据点 yiy_iyi​（见图 222）。若要批判该时间序列的稳定性或离散度，我们最普通的方法是利用到了平均值 yˉ\bar{y}yˉ​ 和（样品）方差 s2s^2s2：

yˉ=1N∑i=1Nyi(1)\bar{y} = \frac{1}{N}\sum_{i=1}^N y_i \tag{1}yˉ​=N1​i=1∑N​yi​(1)

s2=1N−1∑i=1N(yi−yˉ)2(2)s^2 = \frac{1}{N-1}\sum_{i=1}^N(y_i-\bar{y})^2 \tag{2}s2=N−11​i=1∑N​(yi​−yˉ​)2(2)

值得注意一下的是，别忘了样本方差 s2s^2s2 和总体方差 σ2\sigma^2σ2 还是有一些区别的，具体可以参考 CSDN 中的这篇文章。
\quad 用这种方差进行评判，往往适用于 y(t)y(t)y(t) 起伏的过程是稳态的，即统计过程的统计参数如平均值或方差是时间无关的，且数据点 yiy_iyi​ 相互独立的情况。但利用该方差有一个明显的缺点就是：无法细分出不同时间尺度上的误差波动情况。从 111 和图 222 便能粗略看出来，该时间序列值具有短时间的波动项和长时间的漂移项。若我们仅仅粗暴地利用（样品）方差来计算，那只能够反应序列的 “整体情况”，而其中隐含着的短时间波动和长时间漂移等信息则被抹除掉。
\quad 而本文所介绍的 Allan 方差便可以解决这一问题。

# Allan 方差的基本概念

\quad Allan 方差，又称阿兰方差，它是一种能够将不同时间尺度上的不稳定度分门别类地表现出来的一种分析方法，原初就是为了对高精度时钟稳定性进行分析而提出了的。

\quad Allan 方差是将某测量序列在某个指定的时间尺度上的波动情况进行精确的提取，具体做法如下。如图 333 所示，在一次长时间（假若总时长为 TTT）的测量中，设一共测得了 NNN 个数据点（每个数据点的时间间隔为 τ0\tau_0τ0​）。我们按区域长度 τ=(n−1)τ0\tau=(n-1)\tau_0τ=(n−1)τ0​ 将测量序列分为 NcN_cNc​ 块，每一块包含了 nnn 个数据点，区块无重叠。分别计算各区块 nnn 个数据点的均值，记为：

yˉk=1n∑i−kk+n−1yi(3)\bar{y}_k = \frac{1}{n}\sum_{i-k}^{k+n-1}y_i \tag{3}yˉ​k​=n1​i−k∑k+n−1​yi​(3)

利用这 NcN_cNc​ 个阶段性的平均值，计算它们相邻数据的方差的期望值，便可得到 Allan 方差，即：

σy2(τ)=12(Nc−1)∑k=1Nc−1(yˉk+1−yˉk)2(4)\sigma_y^2(\tau) = \frac{1}{2(N_c-1)}\sum_{k=1}^{N_c-1}(\bar{y}_{k+1}-\bar{y}_k)^2 \tag{4}σy2​(τ)=2(Nc​−1)1​k=1∑Nc​−1​(yˉ​k+1​−yˉ​k​)2(4)

Allan 方差 σy2(τ)\sigma_y^2(\tau)σy2​(τ) 的平方根有时也称为 Allan（标准）偏差。Allan 方差以及 Allan 偏差都是基于相邻数值的偏差，而不是像普通方差那样基于与平均值的偏差，这就使得它具有根据选择不同 τ\tauτ ，而反映出不同时间尺度下被测系统不稳定度的能力。简单来说就是：如果你想知道短时间内的波动，即需要较小的 τ\tauτ；若你想看看长时间的漂移，也可以把 τ\tauτ 适当放大。

# 频标领域的噪声类别

\quad 通过改变区块的时间长度 τ\tauτ，并计算对应 τ\tauτ 下的 Allan 偏差，便可以绘出 Allan 偏差随区块长度变化的双对数曲线。在双对数曲线中，有时可以通过它们的斜率来鉴别各种噪声的类型，如图 444 所展示的。

在频标领域，大致有这几种噪声[1]：

  1. σy2(τ)\sigma_y^2(\tau)σy2​(τ) 正比于 τ−2\tau^{-2}τ−2 的相位白噪声（white phase）和闪烁相位噪声（flicker phase）。前者在傅里叶高频区域占比更重，对频标的输出影响可以通过加带通滤波器的办法降低；后者通常由存在噪声的电子线路引起，通过精选元器件可以降低其噪声水平。注意，这两种噪声无法在 Allan 偏差与区域时间 τ\tauτ 的双对数曲线中区分。
  2. σy2(τ)\sigma_y^2(\tau)σy2​(τ) 正比于 τ−1\tau^{-1}τ−1 的频率白噪声（white frequency）。在主动频标中，它可能来源于振荡器回路的热噪声；在被动频标中，它可能来源于光子或原子的散粒噪声，代表了量子极限。
  3. σy2(τ)\sigma_y^2(\tau)σy2​(τ) 与 τ\tauτ 无关的频率闪烁噪声（flicker frequency）。一般出现在主动设备中，例如石英晶体振动器、主动氢钟或激光二极管，而且它也出现在如铯钟等被动频标中。
  4. σy2(τ)\sigma_y^2(\tau)σy2​(τ) 正比于 τ1\tau^{1}τ1 的频率随机游走噪声（random walk frequency）。它通常是因为环境参数影响而产生的，例如温度、振动等。
  5. σy2(τ)\sigma_y^2(\tau)σy2​(τ) 正比于 τ2\tau^{2}τ2 的频率漂移（frequency drift）。可以将它理解为 “随机游走的游走噪声”[2]。

# 光频标的稳定度

\quad Allan 偏差可以用来描述频标系统的稳定度。对于光频标，其稳定度是利用测量的钟跃迁频率的 Allan 偏差来描述的，Allan 偏差值的大小直径反应了钟跃迁频率的抖动大小。
\quad 在光频标领域，Allan 偏差具有关系[3] [4] [5]：

σy(τ)∝1πQs/nTcτ(1N)2(5)\sigma_y(\tau) \propto \frac{1}{\pi Qs/n} \sqrt{\frac{T_c}{\tau}} (\frac{1}{N})^2 \tag{5}σy​(τ)∝πQs/n1​τTc​​​(N1​)2(5)
