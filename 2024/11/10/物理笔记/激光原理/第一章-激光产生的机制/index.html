# 黑体的单色辐射能量密度

\quad 首次学习也许是大二上学期的光学课或者是在大二下学期的热力学与统计物理课，在大三下学期的固体物理里面也是重要的一个知识。在我的一系列笔记中总是那么不厌其烦地提到它：在自由空间，具有任意波矢的单色平面波都可以存在。但在一个有边界条件限制的空间 VVV 内（例如激光器的谐振器，PDH 的超稳腔），只能存在一系列分离的、具有特定波矢 k⃗\vec{k}k的单色平面驻波。每一种不同的 k⃗\vec{k}k称为光波的一种模式。
\quad 我们以一个封闭长方体空腔（如图 111 所示），空腔的体积为 V=ΔxΔyΔzV=\Delta x \Delta y \Delta zV=ΔxΔyΔz。那么沿三个坐标轴方向传播的光波应满足驻波条件：

Δx=mλ2;Δy=nλ2;Δz=qλ2(1.1.1)\Delta x = m\frac{\lambda}{2};\quad\Delta y = n\frac{\lambda}{2};\quad\Delta z = q\frac{\lambda}{2} \tag{1.1.1}Δx=m2λ​;Δy=n2λ​;Δz=q2λ​(1.1.1)

其中 m,n,qm,n,qm,n,q 为正整数。相应的，在波矢空间中，波矢 k⃗\vec{k}k的三个分量应满足条件：

kx=πΔxm;ky=πΔyn;kz=πΔzq(1.1.2)k_x = \frac{\pi}{\Delta x} m;\quad k_y = \frac{\pi}{\Delta y} n;\quad k_z = \frac{\pi}{\Delta z} q \tag{1.1.2}kx​=Δxπ​m;ky​=Δyπ​n;kz​=Δzπ​q(1.1.2)
