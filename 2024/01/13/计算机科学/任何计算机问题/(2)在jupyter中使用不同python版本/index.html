下面将展示任何安装 python3.11.4 版本到 jupyter，并切换使用。

  1. 首先进入 anaconda，输入 jupyter kernelspec list ，查看在 anaconda 中的所有安装的版本。（我是已经安装了 python3.11.4 版本，如果没有安装就不会显示。）

  2. 创建 py3.6.5 虚拟环境，安装该版本 python 到 anaconda3 的 env 目录下（我的目录为 C:\Users\130\anaconda3\envs）。安装利用到 conda create --name py3.11.4 python=3.11.4 命令。其中 name py3.11.4 将命名文件夹， python=3.11.4 指示安装的版本。安装结束后在 env 目录下就会显示 py.3.11.4 的文件

  3. 指令到 py3.11.4 的目录的 Scripts> 下，输入 pip install ipykernel ，就会在 script 目录下会生成一系列工具。

  4. 之后到 C:\Users\130\anaconda3\share\jupyter\kernels 的目录下，对已存在的项目随便复制一份，并重新命名为 python3.11.4。
