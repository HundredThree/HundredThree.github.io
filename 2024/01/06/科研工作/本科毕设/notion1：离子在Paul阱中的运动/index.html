# 宏运动与微运动

\quad 在这里，我们以四杆式线性 Paul\text{Paul}Paul 阱为例，讨论离子在阱中的运动。其结果可以推广到其他类型的 Paul\text{Paul}Paul 阱中。如果有几个离子同时储存在同一个阱中，则这里讨论的运动方程必须修改，但关于微运动及其影响的结论仍然有效。

所以，原子宏运动的振幅 AiA_iAi​ 与特征温度有关，可以通过多普勒冷却来降低特征温度，其实也就是降低动能，以此来降低宏运动的振幅。由 (7)(7)(7) 式可见，宏运动的振幅减小的同时，微运动的动能也相应地减小了。最好的是将振幅冷却为零，这样宏运动和微运动都消失啦！但很不幸，在高等原子分子课程中，我们就学过，多普勒冷却存在一个多普勒冷却极限：

TD=ℏΓ2kB(12)T_{D} = \frac{\hbar\Gamma}{2k_B} \tag{12}TD​=2kB​ℏΓ​(12)

因此宏运动与微运动都是不可能完全补偿掉的。

# 过量微运动

# 杂散静电场引起的过量微运动

\quad 除了上述电极产生的场外，也许还会有些闲杂的均匀静电场 E⃗dc\vec{E}_{\text{dc}}Edc​ 存在，离子受此作用，方程 (6)(6)(6) 变为：

r¨i+[ai+2qicos⁡(ωt)]ω24ri=QE⃗dc⋅e⃗iM(13)\ddot{r}_i + [a_i+2q_i\cos(\omega t)]\frac{\omega^2}{4}r_i = \frac{Q\vec{E}_{\text{dc}}\cdot\vec{e}_i}{M} \tag{13}r¨i​+[ai​+2qi​cos(ωt)]4ω2​ri​=MQEdc​⋅ei​​(13)

该方程的最低阶近似解为：

ri(t)≊(Bi+Aicos⁡(ωit+ψi))[1+qi2cos⁡(ωt)](14)r_i(t) \approxeq (B_i + A_i \cos(\omega_it+\psi_i)) \left[1+\frac{q_i}{2}\cos(\omega t)\right] \tag{14}ri​(t)≊(Bi​+Ai​cos(ωi​t+ψi​))[1+2qi​​cos(ωt)](14)

其中

Bi≊4QE⃗dc⋅e⃗iM(ai+12qi2)ω2≊QE⃗dc⋅e⃗iMωi2(15)B_i \approxeq \frac{4Q\vec{E}_{\text{dc}}\cdot\vec{e}_i}{M(a_i+\frac{1}{2}q_i^2)\omega^2} \approxeq \frac{Q\vec{E}_\text{dc}\cdot\vec{e}_i}{M\omega^2_i} \tag{15}Bi​≊M(ai​+21​qi2​)ω24QEdc​⋅ei​​≊Mωi2​QEdc​⋅ei​​(15)

相比没有杂散静电场下求得的运动方程 (7)(7)(7) 式，(14)(14)(14) 多出了一项 Bi+Biqi2cos⁡(ωt)B_i+\frac{B_iq_i}{2}\cos(\omega t)Bi​+2Bi​qi​​cos(ωt)。其中 BiB_iBi​ 使得粒子偏离了中心点位置 Bxe⃗x+Bye⃗y+Bze⃗zB_x\vec{e}_x+B_y\vec{e}_y+B_z\vec{e}_zBx​ex​+By​ey​+Bz​ez​；而 Biqi2cos⁡(ωt)\frac{B_iq_i}{2}\cos(\omega t)2Bi​qi​​cos(ωt) 则使得粒子额外产生了一个微运动。为了区分上面那个不可避免的微运动，我们将这个称为 “过量微运动”。与微运动不同，过量微运动不能通过冷却方法来显著减小。

# 交流电极相位差引起的过量微运动

\quad 除了杂散电场会导致过量微运动之外，222、444 电极的交流电位之间的相位差 φac\varphi_{ac}φac​ 也可能引起过量微运动。我们设 222、444 电极的电压分别为：

# 过量微运动对冷却极限温度的贡献

\quad 正如 (12)(12)(12) 式告诉我们的，由于宏运动与微运动不可能完全补偿掉，因此会存在一个多普勒极限。举个例子，对于用于 199Hg+^{199}\text{Hg}^+199Hg+ 离子激光冷却的 5d106s2S1/2→5d106p2P1/25d^{10}6s~^{2}S_{1/2}\rightarrow 5d^{10}6p~^2P_{1/2}5d106s 2S1/2​→5d106p 2P1/2​ 跃迁来说，其光谱线宽 Γ=2π×70MHz\Gamma=2\pi\times 70\text{~MHz}Γ=2π×70 MHz。则其激光冷却的多普勒极限为 TD≊1.7mKT_D\approxeq 1.7\text{~mK}TD​≊1.7 mK。
\quad 现在我们把过量微运动也考虑进来，即结合 (14)(15)(20)(14)(15)(20)(14)(15)(20) 式，可得离子沿三个方向的平均动能为：

Eki≊14MAi2(ωi2+18qi2ω2)+4M(qiQE⃗dc⋅e⃗i(2ai+qi2)ω)2+164M(qxRαφacω)2δi,x(21)E_{k_i} \approxeq \frac{1}{4}MA_i^2(\omega_i^2+\frac{1}{8}q_i^2\omega^2) + \frac{4}{M} \left(\frac{q_iQ\vec{E}_{dc}\cdot\vec{e}_i}{(2a_i+q_i^2)\omega}\right)^2 + \frac{1}{64}M(q_xR\alpha\varphi_{ac}\omega)^2\delta_{i,x} \tag{21}Eki​​≊41​MAi2​(ωi2​+81​qi2​ω2)+M4​((2ai​+qi2​)ωqi​QEdc​⋅ei​​)2+641​M(qx​Rαφac​ω)2δi,x​(21)

为了比较上式中后两项相比第一项的贡献，我们将后两项写为 kBTμi/2k_BT_{\mu i}/2kB​Tμi​/2，其中 TμiT_{\mu i}Tμi​ 就是 e⃗i\vec{e}_iei​ 方向上过量微运动所贡献的等价温度。由于 qz=0q_z=0qz​=0，因此沿着轴向方向上的杂散静电场不会改变该方向上的温度。对于一个囚禁在满足 ∣ai∣≪qi2≪1|a_i|\ll q_i^2\ll 1∣ai​∣≪qi2​≪1、ωx=2π×100KHz\omega_x=2\pi\times 100\text{~KHz}ωx​=2π×100 KHz 条件的 Paul 阱中的 199Hg+^{199}\text{Hg}^+199Hg+ 离子，一个大小为 1V/mm1\text{~V/mm}1 V/mm 的均匀 (杂散) 电场会致使 TμxT_{\mu x}Tμx​ 增加 1.4×104K1.4\times 10^4\text{~K}1.4×104 K。若 R=1.0mmR=1.0\text{~mm}R=1.0 mm、α=0.75\alpha=0.75α=0.75，两交流电极相位差 φac=1°\varphi_{ac}=1\degreeφac​=1°，则 TμxT_{\mu x}Tμx​ 增加 1.7×0.41K1.7\times 0.41\text{~K}1.7×0.41 K。这些都远超过了多普勒极限的 1.7mK1.7\text{~mK}1.7 mK。
\quad 而且这些大小的相位差与杂散电场都是可能合理存在的。例如，交流电极的引线可能会由于不同的长度或几何排列而具有不同的电感，因此引起相位差；1V/mm1\text{~V/mm}1 V/mm 的均匀杂散电场可能会因为因为电极不均匀地附着上原子炉内的物质从而产出，也可能会由于电极表面不同晶体平面的贴片效应而产生。
