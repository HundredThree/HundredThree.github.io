# 布洛赫球的定义

qubit\quad\text{qubit}qubit 是俩态量子系统，通常用电子的自旋作为例子，而态空间基底通常记为 ∣0⟩,∣1⟩\ket{0},\ket{1}∣0⟩,∣1⟩。任意一个单 qubit\text{qubit}qubit 态可以被写作：

∣ψ⟩=eiγ(cos⁡θ2∣0⟩+eiϕsin⁡θ2∣1⟩)(3.1.1)\ket{\psi} = e^{i\gamma} \left( \cos\frac{\theta}{2}\ket{0} + e^{i\phi}\sin\frac{\theta}{2}\ket{1} \right) \tag{3.1.1}∣ψ⟩=eiγ(cos2θ​∣0⟩+eiϕsin2θ​∣1⟩)(3.1.1)

其中 θ,ϕ,γ\theta,\phi,\gammaθ,ϕ,γ 都是实数，取值为 0≤θ≤π,0≤ϕ≤2π0\le\theta\le\pi,\ 0\le\phi\le2\pi0≤θ≤π, 0≤ϕ≤2π。而如果不与其他量子系统进行比较的话，γ\gammaγ 毫无测量意义，以此可以将它略去，简化为：

∣ψ⟩=cos⁡θ2∣0⟩+eiϕsin⁡θ2∣1⟩(3.1.2)\ket{\psi} = \cos\frac{\theta}{2}\ket{0} + e^{i\phi}\sin\frac{\theta}{2}\ket{1} \tag{3.1.2}∣ψ⟩=cos2θ​∣0⟩+eiϕsin2θ​∣1⟩(3.1.2)

这是，用 θ,ϕ\theta,\phiθ,ϕ 表征的一个半径为 111 的三维球就称为布洛赫球。其球面的每一点都表征着某个（或不止一个）量子态。
\quad 因此布洛赫球是一个可以将 qubit\text{qubit}qubit 的状态直观化的巧妙工具。

# 布洛赫球的推导

\quad 我们现在就来看看我们是如何导出 (3.1.2)(3.1.2)(3.1.2) 式的。首先对于一般的 qubit\text{qubit}qubit 态我们可以写作：

∣ψ⟩=α∣0⟩+β∣1⟩(3.2.1)\ket{\psi} = \alpha \ket{0} + \beta \ket{1} \tag{3.2.1}∣ψ⟩=α∣0⟩+β∣1⟩(3.2.1)

我们可以将 α,β\alpha,\betaα,β 写成复数的欧拉形式，即：

∣ψ⟩=rαeiϕα∣0⟩+rβeiϕβ∣1⟩(3.2.2)\ket{\psi} = r_\alpha e^{i\phi_\alpha} \ket{0} + r_\beta e^{i\phi_\beta} \ket{1} \tag{3.2.2}∣ψ⟩=rα​eiϕα​∣0⟩+rβ​eiϕβ​∣1⟩(3.2.2)

我们知道，如果不去与其他的量子体系进行比较，单单讨论相干态每一项的相位是没有任何测量意义的，因此我们可以引入一个相位差 ϕ\phiϕ 来代替 ϕα,ϕβ\phi_\alpha,\phi_\betaϕα​,ϕβ​，结果如下：

∣ψ⟩=rα∣0⟩+rβeiϕ∣1⟩(3.2.3)\ket{\psi} = r_\alpha \ket{0} + r_\beta e^{i\phi} \ket{1} \tag{3.2.3}∣ψ⟩=rα​∣0⟩+rβ​eiϕ∣1⟩(3.2.3)

由归一化条件可得：

∣rα∣2+∣rβeiϕ∣2=rα2+rβ2=1(3.2.4)|r_\alpha|^2 + |r_\beta e^{i\phi}|^2 = r_\alpha^2 + r_\beta^2 = 1 \tag{3.2.4}∣rα​∣2+∣rβ​eiϕ∣2=rα2​+rβ2​=1(3.2.4)

因此可将令 rα=cos⁡θ,rβ=sin⁡θr_\alpha=\cos\theta,\ r_\beta=\sin\thetarα​=cosθ, rβ​=sinθ，以此满足上述条件。这样 qubit\text{qubit}qubit 态可以写作：

∣ψ⟩=cos⁡θ∣0⟩+eiϕsin⁡θ∣1⟩(3.2.5)\ket{\psi} = \cos\theta \ket{0} + e^{i\phi} \sin\theta \ket{1} \tag{3.2.5}∣ψ⟩=cosθ∣0⟩+eiϕsinθ∣1⟩(3.2.5)

\quad 或许你觉得上式以及可以画出一个布洛赫球了，但我要告诉你，上式画出的 “布洛赫球” 太过剩余了！我们仔细看看，设一个范围 0≤θ≤π20\le\theta\le\frac{\pi}{2}0≤θ≤2π​，考虑完全球对称的两个点 (1,θ,ϕ),(1,π−θ,ϕ+π)(1,\theta,\phi),\ (1,\pi-\theta,\phi+\pi)(1,θ,ϕ), (1,π−θ,ϕ+π)，前者代表量子态 ∣ψ⟩\ket{\psi}∣ψ⟩，后者代表量子态 ∣ψ′⟩\ket{\psi'}∣ψ′⟩，我们利用 (3.2.5)(3.2.5)(3.2.5) 式可以发现：

∣ψ′⟩=cos⁡(π−θ)∣0⟩+ei(ϕ+π)sin⁡(π−θ)∣1⟩=−cos⁡θ∣0⟩−eiϕsin⁡θ∣1⟩=−∣ψ⟩(3.2.6)\begin{aligned} \ket{\psi'} &= \cos(\pi-\theta) \ket{0} + e^{i(\phi+\pi)} \sin(\pi-\theta) \ket{1} \\ &= -\cos\theta \ket{0} - e^{i\phi} \sin\theta \ket{1} \\ &= -\ket{\psi} \end{aligned} \tag{3.2.6}∣ψ′⟩​=cos(π−θ)∣0⟩+ei(ϕ+π)sin(π−θ)∣1⟩=−cosθ∣0⟩−eiϕsinθ∣1⟩=−∣ψ⟩​(3.2.6)

我们可以看到，两个点所代表的两个态仅仅只相差了一个相因子 −1-1−1。这说明如果用 (3.2.5)(3.2.5)(3.2.5) 式来描绘布洛赫球，上半球面与下半球面所球对称的相应点，都是仅仅只相差了一个相因子 −1-1−1。这样的布洛赫球我们说它是 “剩余的”。

\quad 为此我们可以 “去掉” 下半球面的内容，将上半球面的内容成一个完整球面。这只需要将 (3.2.5)(3.2.5)(3.2.5) 式作 θ→θ2\theta\rightarrow\frac{\theta}{2}θ→2θ​ 这样的变换即可，得到新的布洛赫球表达式为：

∣ψ⟩=cos⁡θ2∣0⟩+eiϕsin⁡θ2∣1⟩(3.2.7)\ket{\psi} = \cos\frac{\theta}{2}\ket{0} + e^{i\phi}\sin\frac{\theta}{2}\ket{1} \tag{3.2.7}∣ψ⟩=cos2θ​∣0⟩+eiϕsin2θ​∣1⟩(3.2.7)

这也就是我们在 (3.1.2)(3.1.2)(3.1.2) 式中所定义的标准布洛赫球表达式。
